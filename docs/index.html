About Me

I'm Maziyar Gerami, a passionate Java backend developer with a strong focus on clean code, software design, and building reliable backend systems. With over 7 years of experience and a deep understanding of the Java ecosystem — including JDK 21, Spring Boot, and Git — I strive to write code that is maintainable, testable, and scalable.

Over the years, I’ve not only developed backend services but also actively contributed to software quality by writing comprehensive unit and integration tests, understanding legacy systems, and improving them iteratively. I am currently open to remote backend roles, especially in teams that value clean architecture, collaboration, and deep focus work. I thrive in environments where feedback is honest, knowledge is shared, and software is crafted with care.
